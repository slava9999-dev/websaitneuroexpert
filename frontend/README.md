# NeuroExpert Frontend

Современный фронтенд для платформы NeuroExpert, построенный на React и Vite.

## Технологии

- **React 18**: Библиотека пользовательского интерфейса.
- **Vite**: Сборщик проектов нового поколения.
- **Tailwind CSS**: Утилитарный CSS-фреймворк для стилизации.
- **Framer Motion**: Библиотека для анимаций.
- **Lucide React**: Набор иконок.
- **Axios**: HTTP-клиент.

## Установка и запуск

1.  **Установка зависимостей**:

    ```bash
    npm install
    ```

2.  **Запуск в режиме разработки**:

    ```bash
    npm run dev
    ```

    Приложение будет доступно по адресу `http://localhost:5173`.

3.  **Сборка для продакшена**:

    ```bash
    npm run build
    ```

    Результат сборки будет в папке `build`.

4.  **Линтинг**:
    ```bash
    npm run lint
    ```

## Структура проекта

- `src/components`: React-компоненты.
- `src/data`: Статические данные (тексты услуг и т.д.).
- `src/services`: API-клиенты (взаимодействие с бэкендом).
- `src/utils`: Утилиты (метрика, видео).

## Особенности

- **Адаптивный дизайн**: Полная поддержка мобильных устройств.
- **Темная тема**: Премиальный дизайн в темных тонах.
- **AI-чат**: Интеграция с умным ассистентом.
- **Оптимизация**: Быстрая загрузка благодаря Vite и оптимизации ассетов.
